Movie Genre Classification using Machine Learning
##Objective
To build a machine learning model that predicts the genre of a movie based on its plot summary using NLP techniques.
 Description
This project uses Natural Language Processing (NLP) to convert text data into numerical format using TF-IDF vectorization. Multiple machine learning models are trained and evaluated to find the best performing model.
##Models Used
- Logistic Regression
- Naive Bayes
- Support Vector Machine
## Workflow
1. Data preprocessing  
2. TF-IDF vectorization  
3. Model training  
4. Model evaluation  
5. Model selection  
##Result
The best model is selected based on highest accuracy and saved for future predictions.
