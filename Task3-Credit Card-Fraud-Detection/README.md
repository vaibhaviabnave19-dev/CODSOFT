# Credit Card Fraud Detection Project

## Objective
To build a machine learning model that detects fraudulent credit card transactions.

## Dataset
- fraudtrain.csv (Training data)
- fraudtest.csv (Testing data)

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest

## Best Model
Random Forest Classifier-81.76%

## Result
Random Forest gave the best performance and was selected as the final model for fraud detection.

- CreditCard_fraud_Detection.ipynb
